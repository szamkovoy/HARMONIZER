# Технический регламент Hume AI

## Параметры аудио (Критично)
- [cite_start]**Частота:** 16000 Hz[cite: 137].
- [cite_start]**Каналы:** 1 (Mono)[cite: 137].
- **Тип данных:** int16 (16-bit PCM).

## Особенности SDK (v.2026)
- [cite_start]**Streaming:** В методе `connect` использовать аргумент `configs` (как список), а не `models`[cite: 138].
- [cite_start]**Результаты:** Эмоции находятся по пути: `result.prosody.predictions[0].emotions`[cite: 139].
- [cite_start]**Топ-эмоции:** Всегда сортировать по `score` в порядке убывания[cite: 139].

## Оптимизация
- Для диагностики эмоционального состояния достаточно сегмента аудио в 10-15 секунд.
